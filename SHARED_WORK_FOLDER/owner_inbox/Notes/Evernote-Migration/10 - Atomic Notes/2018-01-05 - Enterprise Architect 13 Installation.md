STEP 3: INSTALL ENTERPRISE ARCHITECT 

\

To install and register Enterprise Architect Ultimate Edition: 

\

a) If you have installed the trial version of Enterprise Architect, uninstall it using "Add/Remove Programs" in the Windows Control Panel. 

\

b) Double-click on the installation MSI file. The Enterprise Architect Installation Wizard screen displays. Click on the Next button. 

\

c) Read the Licensing Agreement and, if you accept, click on the 'I accept the license agreement' radio button and on the Next button. 

\

d) Read the Readme information, then click on the Next button. 

\

e) Type your User name and Organization names into the appropriate fields, and (optionally) select who can access the application - anyone who uses the computer, or only you (the currently-logged on user). Click on the Next button. 

\

f) If necessary, change the destination folder for the installation, then click on the Next button twice. When the installation is complete, click on the Finish button. 

\

g) Start up Enterprise Architect. The License Management dialog automatically displays. 

\

h) Click on the Add Key button. 

\

i) In the Add Registration Key dialog, select the Enter Private Key tab. 

\

j) Type your User name and Organization names into the appropriate fields, and copy and paste the license key into the Copy registration key... field. 

\

k) Click on the OK button and the Close button to close the License Management dialog. 

\

Your Enterprise Architect Ultimate Edition is now installed and registered!

STEP 5: INSTALL THE MDG ADD-INS 

\

To install and register each MDG Add-In: 

\

a) Double click on the installation EXE file. The MDG Installation Wizard screen displays. Click on the Next button. 

\

b) Read the Licensing Agreement and, if you accept, click on the 'I accept the license agreement' radio button and on the Next button. 

\

c) Read the Readme information, then click on the Next button. 

\

d) Type your User name and Organization names into the appropriate fields, and (Optionally) select who can access the application - anyone who uses the 

computer, or only you (the currently-logged on user). Click on the Next button. 

\

e) If necessary, change the destination folder for the installation, then click on the Next button. 

\

f) (MDG Integration for Eclipse only) If necessary, browse for the correct Eclipse plugins directory path. Click on the Next button. 

\

g) Click on the Next button again. When the installation is complete, click on the Finish button. 

\

h) Repeat this process for the next MDG Add-In to install. 

\

i) Start up Enterprise Architect. Because the MDG Add-Ins are registered under the same license key as Enterprise Architect Ultimate Edition, they are automatically registered and enabled, ready for use!
