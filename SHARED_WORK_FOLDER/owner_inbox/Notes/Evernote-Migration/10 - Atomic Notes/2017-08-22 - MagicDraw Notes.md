- ☐

  **Ctrl-Shift-Q** to make all connections on a diagram rectilinear

- ☐

  **Ctrl-C/Ctrl-Shift-V** to cut/paste visual style from a shape to other shapes

- ☐

  **Selections**

  - ☐

    **Shift-clicking** to add/remove to/from the set of selected objects

  - ☐

    **Alt-clicking** on an object causes all objects of that type to be selected in the current diagram
