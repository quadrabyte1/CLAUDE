Intuit Online (data access notes)

Saturday, December 10, 2011

3:20 PM

**Where to Go From Here**

Every developer of an App Center app must be aware of the access control features described in the topic ["User Access Management API"](http://ipp.developer.intuit.com/0010_Intuit_Partner_Platform/0060_Platform_Services/0020_Platform_API/0003_User_Access_Management_API)

Most applications may need to use some of the app metadata features described in the topic ["Application Management Services API"](http://ipp.developer.intuit.com/0010_Intuit_Partner_Platform/0060_Platform_Services/0020_Platform_API/0004_Application_Management_Services_API).

If your application needs to use database manipulation features, see the topic ["QuickBase for IPP Services API"](http://ipp.developer.intuit.com/0010_Intuit_Partner_Platform/0060_Platform_Services/0020_Platform_API/0007_QuickBase_for_IPP_Services_API). The background topic ["Building and Using Queries"](http://ipp.developer.intuit.com/0010_Intuit_Partner_Platform/0060_Platform_Services/0020_Platform_API/0008_Building_and_Using_Queries) is useful for understanding and using queries whether you're developing a Native IPP Flex app or a Federated app.

Many applications may need to use the file upload and download feature described in the topic ["File Management API"](http://ipp.developer.intuit.com/0010_Intuit_Partner_Platform/0060_Platform_Services/0020_Platform_API/0009_File_Management_API).

IPP Flex apps calling out to Federated web services need to use the features described in the topic ["Federation Services Management API"](http://ipp.developer.intuit.com/0010_Intuit_Partner_Platform/0060_Platform_Services/0020_Platform_API/0006_Federation_Services_Management_API).

Federated IPP applications accessing QuickBooks data via the Intuit Data Services API should review ["Intuit Data Services Management API"](http://ipp.developer.intuit.com/0010_Intuit_Partner_Platform/0060_Platform_Services/0020_Platform_API/0005_Intuit_Data_Services_Management_API).

Error codes that can be returned in the XML responses are provided in ["Error Codes"](http://ipp.developer.intuit.com/0010_Intuit_Partner_Platform/0060_Platform_Services/0020_Platform_API/0010_Error_Codes).
